# Anchor Menu

Anchor Menu - якорное меню для навигации по странице. Для того, чтобы Anchor Menu следовало за скроллом на странице, необходимо для родителя задать свойство `position: relative;`. Anchor Menu имеет возможность изменить положение `top` через кастомное свойство `--ds-anchor-menu-top`.

## Items

ModuleAnchorMenu принимает пропс `items`, который является массивом объектов и предназначен для вывода элементов меню.

Объект массива `items` состоит из следующих полей:

* `href`(*string*\[required]): ID элемента, до которого будет идти прокрутка (пример: #характеристики).
* `tag`(*string*\[optional]): Дополнительная метка справа от элемента меню.
* `text`(*string*\[required]): Текст элемента меню.
