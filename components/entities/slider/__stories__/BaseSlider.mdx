# Slider

Slider — это динамический блок страницы, который показывает несколько элементов контента по очередности.

## Применение

Slider можно использовать для вывода фотографий или карточек с информацией, которые меняются через определенный промежуток времени.

## Варианты использования

## Autoplay

Slider принимает пропс `autoplay`, который устанавливает время через которое произойдет перелистывание на следующий слайд. Время задается в `ms`. По умолчанию автоплей выключен.

## Breakpoints

Slider принимает пропс `breakpoints`, который позволяет установить ширину экрана для которой будет манятся конфигурация слайдера. По умолчанию `breakpoints` не задан.

## Navigation

Slider принимает пропс `navigation`, который скрывает стрелки переключения слайдов. По умолчанию стрелки отображается.

## Slides

Slider принимает пропс `slides`, который является массивом слайдов.

## Slides per view

Slider принимает пропс `slidesPerView`, который устанавливает число отображаемых слайдов за раз. По умолчанию выводит один слайд за раз.

## Pagination

Slider принимает пропс `pagination`, который скрывает пагинацию. По умолчанию пагинация отображается.
