# Form

Form — представляет собой раздел документа, содержащий интерактивные элементы управления для отправки информации.

<Canvas of={BaseFormStories.Overview} />

<Controls story="Overview" />

## Применение

Form используется для сбора информации, внесенной пользователем.

## Варианты использования

## Slot

Form может принимать `slot`

<Canvas of={BaseFormStories.Slot} />

### Title

Form принимает атрибут `title` для установки заголовка формы. По умолчанию нет заголовка.

<Canvas of={BaseFormStories.Title} />

### Button

Form принимает атрибут `button` для установки заголовка и иконки кнопки.
`button` является объектом содержащий поля `icon` и `label`.

```js
{
  icon: 'chevron-right', // Значение по умолчанию для иконки
  label: 'Отправить', // Значение по умолчанию для заголовка кнопки
}
```

<Canvas of={BaseFormStories.Button} />

### Button disable

Form принимает атрибут `buttonDisable` для установки `disable` состояния кнопки. По умолчанию кнопка находится в `disable` состоянии.

<Canvas of={BaseFormStories.ButtonDisable} />

### Description

Form принимает атрибут `description` для установки описания формы. По умолчанию описание отсутствует.

<Canvas of={BaseFormStories.Description} />

### Validation shema

Form принимает атрибут `validationSchema` для установки валидации формы.
Валидация полей может устанавливается с помощью библиотеки [yup](https://www.npmjs.com/package/yup).

<Canvas of={BaseFormStories.ValidationSchema} />

### Background

Form принимает атрибут `backgroundImg` для установки фона.

### Position

Position может иметь один из двух вариантов: `center` и `right`. По умолчанию `center`.

### Slots

Form принимает следующие слоты: `default`, `actions`.

#### Default

Является телом формы и может содержать к примеру различные инпуты, подсказки и текст.

#### Actions

Слот предназначен для переопределения кнопок или расширения функцианала кнопок.
Слот предоставляет доступ к тем же реквизитам, что указаны в документации [vee-validate](https://vee-validate.logaretm.com/v4/api/form#slots)
