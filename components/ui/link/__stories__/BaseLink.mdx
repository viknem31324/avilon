## Варианты использования

### Size

Link может наследовать размер шрифта блока, в котором находится, или иметь один из трех размеров: `s`, `m` и `l`. По умолчанию наследует размер шрифта.

<Canvas of={BaseLinkStories.Sizes} />

### Variant

Устанавливает вариант ссылки. Может иметь один из двух значений: `green`, `black`. По умолчанию `green`
