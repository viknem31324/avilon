## Items

RhsTags принимает пропс `items`, который является массивом объектов и предназначен для вывода списка тегов.
При большом количестве тегов, часть из них скрывается и доступна при нажатии на кнопку показа или через дропдаун

Объект массива `items` состоит из следующих полей:

* `id`(*string | number*\[required]): ID тега. Указывается в аргументах события клика по тегу.
* `label`(*string*\[required]): Текст тега.
* `active`(*boolean*\[optional]): Флаг, указывающий, активен ли тег. Активный тег имеет другие цвета текста, иконки и фона.
* `icon`(*string*\[optional]): Название иконки тега. Отображение иконки зависит от варианта тега.
* `link`(*string*\[optional]): url адрес ссылки. При наличии данного атрибута, тег будет выводить внутри себя ссылку с указанным url.

## Применение

Tags используется для элементов, которые необходимо пометить, классифицировать или организовать с помощью ключевых слов, которые их описывают.
