# Player

Player - компонент для отображения видео. Через поле `video` можно задавать ссылку для видео. Ссылка может вести как на видео c `YouTube`, `Rutube`, так и на файл поддерживающий видеоформат (к примеру `.mp4`, `.mov` и прочие).
