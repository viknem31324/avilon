# Radiobutton

Radiobutton позволяет пользователям выбирать один элемент из набора опций.

### Size

Устанавливает размер.

Radiobutton может иметь один из трех размеров: `s`, `m` и `l`.

<Canvas of={BaseRadioButtonStories.Size} />

### Label

Устанавливает подпись.

Radiobutton может использоваться с подписью и без нее.

<Canvas of={BaseRadioButtonStories.Label} />

## Error

Компонент `BaseRadioGroup` поддерживает состояние ошибки.

Текст ошибки можно задать во время описания правил валидации передаваемых в `validationSchema`.
